# IEEE-754 Binary-32 Floating Point Translator

This is a simple web program that can convert between binary, hexadecimal, and decimal representations of IEEE-754 Binary-32 floating point numbers.

## Instructions

Follow these steps to use the conversion tool:

### 1. Choose Input Type

Select between two input types:

- **8-Digit Hexadecimal**
- **32-Bit Binary**

<img src="https://github.com/user-attachments/assets/1429755f-69ec-4f27-8d94-b118ec7d9ba1" width="600" alt="Input Type Selection">

### 2. Input the Number

Enter the number based on the selected input type:

<img src="https://github.com/user-attachments/assets/68e520a5-1ea3-40fc-a67f-7ea72a6901b2" width="600" alt="Input Number">

### 3. Choose Decimal Output Type

Select between two decimal output types:

- **Fixed Point**
- **Floating Point**

<img src="https://github.com/user-attachments/assets/056b4655-c6de-4e0c-8013-99b9fc476bda" width="600" alt="Decimal Output Type">

### 4. Convert the Number

Click the **"Convert"** button and wait for the result:

<img src="https://github.com/user-attachments/assets/b8e780b2-6fe5-4a78-a503-d11dc8286c1b" width="600" alt="Convert Button">
